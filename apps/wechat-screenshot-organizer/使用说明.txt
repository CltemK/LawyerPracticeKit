微信聊天记录截图整理工具 - 使用说明
======================================

一、软件功能
------------
本工具用于自动整理微信聊天记录截图，通过OCR技术识别时间戳，按时间顺序生成规范的Word文档。

主要功能：
- 自动识别截图中的时间信息
- 按时间顺序自动排序图片
- 批量处理多张截图
- 生成格式统一的Word文档
- 支持时间戳标注


二、使用方法
------------

1. 启动软件
   双击 "微信截图整理工具（完整版）.exe"

2. 配置OCR引擎（首次使用）
   点击右上角【⚙️ 设置】按钮：

   推荐选择【PaddleOCR（本地）】：
   - 永久免费，无需联网（完整版已包含模型）
   - 图片本地处理，保护隐私
   - 需要安装VC++运行库（仅首次）

   备选【百度OCR（在线API）】：
   - 需要联网和API密钥
   - 无需VC++运行库
   - 免费额度：50000次/天

3. 添加截图
   - 点击"📁 上传图片"选择单张或多张截图
   - 或点击"📂 选择文件夹"批量导入

4. 调整设置
   - 每页图片数量：1-9张
   - 显示时间戳：是否显示识别的时间

5. 生成文档
   - 点击"✨ 生成Word文档"
   - 选择保存路径
   - 等待处理完成


三、支持的时间格式
------------------
应用能够识别以下微信聊天记录时间格式：

- 2024年4月16日 中午12:57
- 2024年4月16日 下午13:02
- 昨天 15:30
- 今天 上午10:30
- 4月16日 下午13:02


四、OCR引擎对比
---------------

| 特性 | 百度OCR | PaddleOCR（完整版） |
|------|---------|---------------------|
| 费用 | 免费额度后收费 | 永久免费 |
| 联网 | 需要联网 | 无需联网 |
| 识别速度 | 快（云端计算） | 中等（本地计算） |
| 识别准确率 | 高 | 高 |
| 配置难度 | 需要申请 API 密钥 | 安装即用 |
| 隐私 | 图片上传到云端 | 图片本地处理，更安全 |
| 首次使用 | 直接可用 | 完整版无需下载模型 |

推荐：使用 PaddleOCR（完整版），配置简单，永久免费，已包含模型文件。


五、系统要求
------------
- Windows 10/11 64位系统
- 2GB以上内存
- 100MB以上磁盘空间


六、注意事项
------------
1. 使用PaddleOCR需要安装VC++运行库（vc_redist.x64.exe）
2. 图片文件大小不应超过4MB（应用会自动压缩）
3. 建议截图清晰度足够，以提高OCR识别准确率
4. 配置文件和密钥文件会保存在应用目录下


七、常见问题
------------

Q: 应该选择哪个OCR引擎？
A: 推荐选择 PaddleOCR（完整版），已包含模型文件，无需联网下载。

Q: PaddleOCR报错怎么办？
A: 1. 确保已安装VC++运行库
   2. 删除C:\Users\你的用户名\.paddleocr文件夹后重试
   3. 或切换到百度OCR

Q: OCR识别失败怎么办？
A: 请检查网络连接（百度OCR需要联网）或API密钥配置。

Q: 时间识别不准确怎么办？
A: 应用会提取截图中所有时间戳，使用最早时间排序。如果识别失败，图片会按文件名排序。

Q: 可以自定义文档布局吗？
A: 可以在底部设置每页图片数量（1-9张），应用会自动计算网格布局。


八、技术支持
------------
如有问题，请检查：
1. 是否已安装VC++运行库（使用PaddleOCR时）
2. config.json 配置文件是否正常生成
3. 软件目录是否有写入权限
